==================================================
📋 전체 프로젝트 요약 (프론트엔드 + 백엔드)
==================================================

✅ 1. 프로젝트 개요
--------------------------------------------------
• 목표
  - 사용자 결제/환불 기능과 관리자 대시보드를 포함한 통합 예약 관리 시스템 구축

• 구성
  - Frontend: React 기반 사용자/관리자 화면
  - Backend: Spring Boot 기반 결제, 환불, 예약, 멤버십, 통계 API 서버

• 주요 기술
  - Frontend: React, React Query, Axios, TailwindCSS, Recharts
  - Backend: Spring Boot, Spring Data JPA, Spring Security, Iamport API, ModelMapper


✅ 2. 프론트엔드 주요 기능
--------------------------------------------------
• 사용자 결제 & 환불
  - 객실 및 서비스 예약 후 결제
  - 결제 요약 및 할인 정보 제공
  - 환불 요청 및 진행 상태 확인

• 관리자 대시보드
  - 전체 결제 내역 조회
  - 환불 승인 처리
  - 예약 리스트 관리
  - 통계 시각화 (환불률, 총 결제 금액, 인기 객실, Top 사용자)

• 기술 포인트
  - React Query를 활용한 데이터 캐싱 및 갱신
  - Axios Interceptor를 통한 JWT 인증 자동화
  - Lazy Loading 최적화
  - Recharts로 통계 데이터 시각화


✅ 3. 백엔드 주요 기능
--------------------------------------------------
• 결제 및 환불 관리
  - Iamport API 연동을 통한 결제 승인/환불 처리
  - 결제 → 주문 생성 → 환불 시 상태 업데이트
  - 멤버십 할인율 적용

• 주문 및 예약 관리
  - 사용자 주문 조회 및 환불 요청
  - 관리자의 환불 승인 및 자동 환불
  - 예약 등록, 수정, 삭제 API

• 멤버십 관리
  - 멤버십 생성, 수정, 삭제, 사용 처리
  - 누적 결제액에 따른 등급 자동 승급 (GOLD → DIAMOND → TRINITY)

• 관리자 통계
  - 총 결제 금액, 총 환불 금액, 환불률, 인기 객실, Top 사용자 랭킹 제공

• 기술 포인트
  - Spring Security를 통한 STAFF 권한 검증
  - RestTemplate + IamportClient를 통한 외부 결제 서버 연동
  - ModelMapper를 통한 DTO 변환
  - JPA + JPQL 맞춤형 쿼리 작성


✅ 4. 주요 데이터 흐름 요약
--------------------------------------------------
• 사용자 결제 흐름
  객실 선택 → 결제 페이지 → 결제 생성 → 결제 승인 → 주문 생성 → 마이페이지 이동

• 사용자 환불 흐름
  마이페이지 → 환불 요청 → 관리자 승인 → 결제 취소 → 환불 완료

• 관리자 통계 흐름
  전체 주문/결제 데이터 집계 → 환불률, 인기 객실, Top 사용자 랭킹 계산 → 대시보드 표시


✅ 5. 한 문장 요약
--------------------------------------------------
"사용자와 관리자가 모두 사용할 수 있는 결제·환불·예약 통합 관리 시스템을 
React + Spring Boot로 설계부터 구축까지 직접 완성했습니다."


✅ 6. 특별히 강조할 수 있는 포인트 ✨
--------------------------------------------------
• 결제 시스템과 통합된 멤버십 할인 및 자동 승급 로직 구현
• 환불 요청 → 승인 → 아임포트 API 연동까지 환불 프로세스 완성
• 관리자 대시보드에서 실시간 통계 시각화 제공
• JWT 인증 자동화, React Query 데이터 동기화 최적화 적용
